# API-El-Geladon
Aplicação consistirá em um CRUD completo no backend em nodeJs, utilizando o framework express, além de introduzir novos aspectos como as camadas de routes e service que, em conjunto com o controller.

### Stacks

Módulos utilizados:
*JavaScript
*Express
*Nodemon
*MongoDB
